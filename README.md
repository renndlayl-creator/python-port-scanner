# Python Port Scanner

Scanner de puertos TCP hecho en Python.

## Funciones
- Escaneo de múltiples puertos
- Detección de puertos abiertos
- Uso de sockets TCP
- Configuración de timeout
- Escaneo local con localhost

## Tecnologías
- Python
- Socket

## Cómo usar

Ejecutar:

```bash
python scaner.py
